# Projeto Bora — Fase 1: Arquitetura do Produto e do SaaS

A Bora será construída como um SaaS simples, personalizável e em tempo real para pequenos deliveries controlarem pedidos, entregas e vendas.

## Estrutura geral aprovada
1. Fase 1 — Arquitetura do produto e do SaaS
2. Fase 2 — Front-end e experiência do usuário
3. Fase 3 — Back-end Java, banco de dados e integrações
4. Finalização — Testes, documentação, deploy e plano comercial

## Tecnologia alvo
- Back-end: Java 21 + Spring Boot 3
- Banco: PostgreSQL
- Migrações: Flyway
- Segurança: Spring Security + JWT
- Front-end: HTML, CSS e JavaScript inicial, pronto para evoluir para React
- Deploy: Docker
