# Projeto Bora — Pacote Completo

A Bora é um SaaS simples, personalizável e em tempo real para pequenos deliveries controlarem pedidos, entregas e vendas.

Este pacote consolida as fases:
1. Arquitetura do produto e do SaaS
2. Front-end e experiência do usuário
3. Back-end Java, banco de dados e integrações
4. Finalização: testes, documentação, deploy e plano comercial

## Stack
- Java 21 + Spring Boot 3
- PostgreSQL + Flyway
- HTML/CSS/JavaScript no front-end inicial
- Docker para deploy
